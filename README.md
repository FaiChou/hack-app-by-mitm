## Node.js version

#### Install

```
$ yarn
```

#### Usage

```
$ yarn start 
```

## Python3 version

#### Install

```
$ brew install mitmproxy
```

#### Usage

```
$ mitmproxy -s hack-sip.py
```

Or just using script:

```shell
$ hack_sip
```

Enjoy!
